Array Exercises

  • 1. Write a JavaScript function to check whether an `input` is an array or not.
  • 2. Write a JavaScript function to clone an array.
  • 3. Write a JavaScript function to get the first element of an array. Passing a parameter 'n' will return the first 'n' elements of the array.
  • 4. Write a JavaScript function to get the last element of an array. Passing a parameter 'n' will return the last 'n' elements of the array.
  • 5. Write a simple JavaScript program to join all elements of the following array into a string.
  • 6. Write a JavaScript program which accept a number as input and insert dashes (-) between each two even numbers. For example if you accept 025468 the output should be 0-254-6-8
  • 7. Write a JavaScript program to sort the items of an array.
  • 8. Write a JavaScript program which accept a string as input and swap the case of each character. For example if you input 'The Quick Brown Fox' the output should be 'tHE qUICK bROWN fOX'.
  • 9. Write a JavaScript program to compute the sum and product of an array of integers.
  • 10. Write a JavaScript program to add items in an blank array and display the items.
  • 11. Write a JavaScript program to remove duplicate items from an array (ignore case sensitivity).
  • 12. Write a JavaScript program to shuffle an array.
  • 13. Write a JavaScript program to perform a binary search
  • 14. Write a JavaScript program to find duplicate values in a JavaScript array.
  • 15. Write a JavaScript function to sort the following array of objects by title value.
  • 16. Write a JavaScript function to merge two arrays and removes all duplicates elements.
  • 17. Write a JavaScript function to remove a specific element from an array.
  • 18. Write a JavaScript script to empty an array keeping the original.
  • 19. Write a JavaScript function to get nth largest element from an unsorted array.
  • 20. Write a JavaScript function to get a random item from an array.
  • 21. Write a JavaScript function to create a specified number of elements with pre-filled numeric value array.
  • 22. Write a JavaScript function to move an array element from one position to another.
  • 23. Write a JavaScript function to filter false, null, 0 and blank values from an array.
